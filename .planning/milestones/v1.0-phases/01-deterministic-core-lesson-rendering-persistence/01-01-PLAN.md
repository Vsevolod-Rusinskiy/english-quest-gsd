This plan delivers the walking-skeleton slice of that goal: theory + the first text-input exercise, end to end, persisted.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| static asset → app (`fetch('/Lesson-1A.json')`) | Lesson JSON is untrusted input at the parse boundary; a malformed/tampered file must fail loudly, never render partially (ASVS V5) |
| localStorage → app (`load()`) | localStorage is user-editable via devtools; a tampered/corrupt progress blob must reset to a fresh state, never crash (OWASP: localStorage content is not trusted) |
| lesson-derived text → DOM render | Any string from lesson JSON rendered into the DOM is a potential stored-XSS vector if injected via innerHTML |
